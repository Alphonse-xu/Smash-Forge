Smash Forge

About Download Source Wiki

Game-Accurate Model Rendering

Smash Forge can import and view multiple smash model formats including Namco Model Format and Melee's .dat files. This can be used in conjunction with a number of other features, displays imported models and auto-updates with changes to materials, textures, animations and bones.

Real Time Hitbox Visualization

Forge has the ability to visualize the hitboxes of every move for every character in smash 4 in realtime, frame by frame with or without models, color coded to show hitbox information at a glance. Designed for both rapid moveset development and competitive smash 4 analysis.

Making Level Data Editing Easier Than Ever

Display the level data right on top of the stage models, allowing you to edit the level data to match perfectly the first time or simply to learn more about every aspect of stages to improve your game knowledge. From creating a stage from scratch to tweaking the boundaries of existing stages, Forge can help make every stage the way you want.

Built in Texture Manager

Use the NUT editor to manage, import and ID textures, all in one place. Import DDS or PNG textures, check mipmaps and assign IDs for your textures to assign them to your models in a simple and efficient manner.

Easiest Way to Modify Smash

Forge has a built-in parameter editor for modifying Smash 4's param files, changing things from the Character Select Screen to Classic Mode Events or even fine tune the base mechanics of the game. Includes community contributed labels for many of the most important param files, removing the need to reverse engineer or double check what parameters mean.
